React é uma biblioteca em JavaScript, por isso é necessário conhecer JavaScript. Não é necessário conhecer Node, mas para usar as ferramentas necessárias para usar React precisam de ter Node instalado. Algumas funcionalidades desta nova sintaxe são necessárias para usar React, como por exemplo as classes e os módulos.

React

"A JavaScript library for building user interfaces."

Made by Facebook

Just the UI

Makes no assumptions about the rest of the application.

Super simple

Doesn't have a huge list of features.

Fast to start using.

"Learn once, write anywhere"

React was built for the web, but can also be used to create mobile or desktop application.

Requirements

  • Package manager: Npm or Yarn
  • Bundler: Webpack or Browserify
  • Compiler: Babel

Components

React is all about writing components.

They let you split your application into small pieces.

Props

A component can receive properties.

The properties are passed down by the parent.

State

The state is updated by the component itself.

When the state is updated, the view will re-render itself.

Only class components can have state.

Children

A component can receive children.

Children work as props.

Component methods and fields

render()

Must return a React element, that can be another component or a native DOM element (like a <div>).

The render method is required.

Styling

Components can be stiled like normal HTML elements, with CSS classes or inline styles.
