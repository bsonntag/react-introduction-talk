React é uma biblioteca em JavaScript, por isso é necessário conhecer JavaScript. Algumas funcionalidades desta nova sintaxe são necessárias para usar React, como por exemplo as classes e os módulos.

Web development

React

"A JavaScript library for building user interfaces."

Made by Facebook

"Learn once, write anywhere"

Components

React lets you split your application into small pieces,
called components.

Component properties

Information can be passed to a component using properties.

Component state

Information can be stored in a component's state.

Updating the UI

the React will update the interface.
